# <a name="reference">@lcf.vs/dom</a>

Some utility classes, as ES6 module, to easily the DOM, even in front/back

## <a name="install">Install</a>

`npm i @lcf.vs/dom`

## <a name="usage">Usage</a>

```js
import dom from '@lcf.vs/dom'

console.log(dom)
```
